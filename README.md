# WebCraft Studio Portfolio

A premium responsive portfolio website built for showcasing demo websites for local businesses such as restaurants, clinics, salons, bakeries, cafés, and rental companies.

## Live Features

- Premium 3D animated hero section
- Responsive modern UI
- Live demo showcase section
- Mobile-first design
- Reusable navbar and footer
- Smooth animations and reveal effects
- Business-focused layouts
- Contact & WhatsApp integration ready

## Demo Websites

- Restaurant Website
- Barber Shop Website
- Dental Clinic Website
- Luxury Car Rental Website
- Bakery / Café Website

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Three.js
- GLTFLoader
- EmailJS
- Vercel Deployment

## Deployment

Hosted using Vercel.
